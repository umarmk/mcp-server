# MCP-Server

A lightweight JSON-RPC (MCP) server exposing CRUD operations over PostgreSQL.

## Tech Stack

- Python 3.10+
- FastAPI
- jsonrpcserver
- asyncpg / SQLAlchemy (async)
- Pydantic
- Uvicorn

## Getting Started

1. Clone the repo
2. Create and activate a virtual env
3. Install dependencies
4. Run `uvicorn server:app --reload`
